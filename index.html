Welcome to the Student Website. 


This website is developed to provide complete and accurate information about
a student in a simple and user-friendly manner. The purpose of this website
is to organize student details such as personal information, academic
subjects, and contact details in one place.

The website contains multiple sections including Home, About, Subjects,
and Contact. Each section is designed to help users easily navigate and
understand the information available. The About section provides personal
and educational details of the student, the Subjects section displays the
list of subjects studied, and the Contact section allows communication with
the student.

This website is created using HTML and CSS as a part of learning web
designing and front-end development. It helps in understanding how to
create structured web pages, apply styling, and link multiple pages
together.
