About This Website

        This website is created as a student project to understand the basics of
web designing and front-end development. It is designed to provide useful
information to students related to stationery items and academic subjects.

The main aim of this website is to demonstrate the use of HTML for page
structure, CSS for styling, and JavaScript for basic form validation and
navigation.

This project helps students learn how to create multi-page websites,
connect pages using navigation links, and design simple user-friendly
interfaces.
